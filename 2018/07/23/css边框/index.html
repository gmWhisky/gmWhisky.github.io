css边框

发表于 2018-07-23

css边框

  • boder
  • boder-color
  • boder-style
  • boder-width
  • boder的上下左右

boder-style:设置四个边框的样式

例如： p{boder-style:solid;}
这个属性的一些可能的值：
hidden ：用于解决边框冲突 
dotted : 点状边框
dashed : 定义虚线
solid : 定义实线
double : 定义双线
groove : 定义3D凹槽边框
ridge : 定义3D垄状边框
inset : 定义3Dinset边框
outset ：定义3Doutset边框

需要注意的是：还可以为一个边框定义多个样式
例如 p {boder-style :solid dotted dashed double;}
这里的值采用了 top-right-bottom-left 的顺序
如果你想为元素框的某一个边设置样式

  • boder-top-style
  • boder-right-style
  • boder-bottom-style
  • boder-left-style

还有就是单边属性必须放在简写属性之后，否则会被覆盖掉

boder-width:为边框指定宽度

例如: p{boder-style:solid;
        boder-width:5px;}
这个属性运用有两种方法：
1 指定长度值 ：2px 0.1em
2 使用三个关键字 ：thin medium(默认) thick

同样的 这个也可以定义单边宽度 按照top-right-bottom-left 的顺序
例如 ：p {border-style: solid; border-width: 15px 5px 15px 5px;}

如果你想为元素框的某一个边设置宽度

  • boder-top-width
  • boder-right-width
  • boder-bottom-width

  • boder-left-width

    如果把border-style设置为none，表示边框不存在就没有宽度了，其次，不要忘记声明边框样式，因为默认值为none

boder-color:边框的颜色

例如：p{boder-style:solid;
boder-color:blue rgb(25%,35%,45%) #909090 red;}

如果颜色值少于4个，值复制就会起作用
p{border-color: blue red;}
表示的是上下边框为蓝色 左右边框为红色

如果你想为元素框的某一个边设置颜色

  • boder-top-color
  • boder-right-color
  • boder-bottom-color

  • boder-left-color

    如果要创建有宽度的不可见边框
    就加入 boder-color: transparent;
