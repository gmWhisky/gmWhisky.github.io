三栏布局

发表于 2018-07-27

圣杯布局和双飞翼布局

圣杯布局和双飞翼布局都是为了实现两边盒子宽度固定，中间盒子自适应的三栏布局，也就是我们常说的固比固布局，实现的效果是一样的，差别在于思想。

通过缩放页面，随着页面宽度的变化，三栏布局是中间盒子优先渲染，两边盒子框子固定不变，即使页面宽度变小，不影响浏览。
注意：当你缩放页面的时候，宽度不能小于700px，所以，最好还是给body加最小宽度min-width

圣杯布局

HTML结构

写结构的时候要注意，父元素的的三栏务必先写中间盒子。因为中间盒子是要被优先渲染嘛~并且设置其自适应，也就是width:100%

css样式

利用负变局布局
1 让左边的盒子上去

需要设置其左边距为负的中间盒子的宽度，也就是.left {margin-left:-100%;}。这样左盒子才可以往最左边移动。

2 让右边的盒子上去

需要设置其左边距为负的自己的宽度，也就是.right {margin-left：-200px;}。这样右盒子才可以在一行的最右边显示出自己。

3 让中间自适应的盒子显示

先是 利用父级元素将盒子往中间挤

代码如下：.container{ padding: 0 200px;} 这里的200px是左右盒子的宽度。

然后 给左右盒子加定位

代码如下：
.left{ position: relative; left: -200px;}
.right{position: relative;right: -200px;}

利用负变局布局
1 让左边的盒子上去

需要设置其左边距为负的中间盒子的宽度，也就是.left {margin-left:-100%;}。这样左盒子才可以往最左边移动。

2 让右边的盒子上去

需要设置其左边距为负的自己的宽度，也就是.right {margin-left：-200px;}。这样右盒子才可以在一行的最右边显示出自己。

双飞翼和圣杯的差别就在这里，当middle里面再嵌套一个盒子后，就可以直接设置margin来调整宽度，而不影响外边的盒子，两边的就不用再定位了

代码

.middle-in{margin: 0 200px; }
